\# AI Model Inference and Prompt Engineering Using Ollama



\## Student



\*\*Eilene Anna Kuriakose\*\*



\## Project Overview



This project demonstrates prompt-engineering techniques and AI model inference using Ollama and the Llama 3.2 model.



The project includes zero-shot prompting, one-shot prompting, few-shot prompting, chain-of-thought prompting, text generation, text summarization, question answering, concept-level processing, and an implementation of the Segment Anything Model (SAM).



\## Features



\- Local LLM inference using Ollama

\- Llama 3.2 model integration

\- Installed Ollama model listing

\- Zero-shot prompting

\- One-shot prompting

\- Few-shot prompting

\- Chain-of-thought prompting

\- Next-text generation

\- Text summarization

\- Question answering

\- LCM-style concept-level processing

\- SAM image-segmentation implementation

\- Inference-time measurement

\- JSON result generation

\- TXT result generation

\- Error handling



\## Technologies Used



\- Python

\- Ollama

\- Llama 3.2

\- Segment Anything Model
